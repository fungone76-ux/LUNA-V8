## ⚙️ CONFIGURAZIONE AVANZATA

### **Modificare Stack Iniziale**

In `poker_game.py`, quando crei il gioco:
```python
poker = PokerGame(
    engine=self.engine,
    companion_names=["Luna"],
    initial_stack=2000,  # ← Cambia qui (default 1000)
)
```

**Effetto**:
- Stack più alto = più mani prima dello strip
- Stack più basso = strip più veloce

**Consigliato**:
- 500 chips = Partita veloce (~10 mani)
- 1000 chips = Partita normale (~20 mani)
- 2000 chips = Partita lunga (~40 mani)

---

In `simple_strip_manager.py`, modifica `LEVELS`:


## 📊 PERFORMANCE NOTES

### **Image Generation Time**

Ogni strip level genera 1 immagine:
- Tempo medio: 5-10 secondi per immagine
- Partita completa (5 levels): 25-50 secondi totali

**Ottimizzazione**:
- Cache immagini già generate
- Pre-generate common scenarios
- Opzione "skip images" per testing

### **Memory Usage**

- Poker game state: ~5KB
- Strip manager: ~2KB per companion
- Total per game: ~15KB (trascurabile)
